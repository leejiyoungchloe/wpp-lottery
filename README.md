# WPP Media Delivery 618 Kick Off 抽奖本地版

## 启动

### 普通局域网模式

```bash
npm start
```

### 公网隧道模式（手机使用移动数据也能扫码）

```bash
npm run start:tunnel
```

或双击 `start-with-tunnel.command`（macOS）/ `start-with-tunnel.bat`（Windows）。

启动后终端会显示类似：

```text
http://127.0.0.1:5173
http://10.x.x.x:5173
```

大屏建议打开 `http://127.0.0.1:5173` 或终端里显示的局域网地址。页面二维码会优先使用局域网地址，手机和电脑需要在同一个网络里。

## 功能

- 扫码登记姓名，也支持现场手工添加和批量导入。
- 参与人员总数实时显示。
- 一等奖 1 名、二等奖 2 名、三等奖 3 名，按奖项分批抽取。
- 每个奖项滚动 20 秒后停止并产出该奖项名单。
- 已中奖人员不会进入后续奖项。
- 后端数据保存在 `data/lottery-data.json`，刷新页面不会丢。
- 可导出参与名单 CSV。

## 微信姓名

普通微信扫码打开网页不能直接读取用户真实姓名。要自动带出微信姓名，需要接入公众号 OAuth、企业微信 OAuth，或提前生成带姓名参数的专属链接。当前本地版默认让用户扫码后手动填写姓名。
